Python File Handling: The `open()` Function

The key function for working with files in Python is the open() function. It takes two parameters: filename and mode.

There are four main modes for opening a file:

  • "r" - Read (Default). Error if file doesn't exist.
  • "a" - Append. Creates file if it doesn't exist.
  • "w" - Write. Overwrites content; creates file if needed.
  • "x" - Create. Creates file; error if it exists.

You can also specify "t" (Text - Default) or "b" (Binary - e.g., images).

Real-World Scenario
A logging system needs to open a file. It uses "a" (Append) to add new error logs without deleting old ones, or "w" to rotate the log file daily.
modes.py
# Syntax: open(filename, mode) # "r" - Read - Default value # f = open("demofile.txt") # The code above is same as: # "r" for read, "t" for text are defaults f = open("demofile.txt", "rt") # "b" - Binary mode (e.g. images) # img = open("image.jpg", "rb")

Opening & Reading Files

reading.py
# Open file located in same folder f = open("demofile.txt") # The open() function returns a file object print(f.read()) # Open file on a different location (Windows) f = open("D:\\myfiles\\welcome.txt") print(f.read())

To open a file for reading, it is enough to specify the name of the file. The open() function returns a file object, which has a read() method for reading the content.

If the file is in a different location, you must specify the file path. On Windows, you must use double backslashes \\ to escape the path.

!
Pro Tip: Make sure the file exists before opening in "r" mode, otherwise Python will raise a FileNotFoundError.

The `with` Statement & Closing Files

It is good practice to always close the file when you are done with it. If not, changes may not show due to buffering.

However, the best practice is to use the with statement. It automatically closes the file even if the script encounters an error.

Real-World Scenario
Processing a financial transaction file. If the script crashes halfway, the `with` statement ensures the file handle is released so other processes can access it.
context.py
# RECOMMENDED: Using 'with' statement with open("demofile.txt") as f: print(f.read()) # File closes automatically here # MANUAL: Close the file explicitly f = open("demofile.txt") print(f.readline()) f.close() # Must remember to call this!

Reading Parts & Lines

read_advanced.py
# Read only the first 5 characters with open("demofile.txt") as f: print(f.read(5)) # Read one line with open("demofile.txt") as f: print(f.readline()) # Loop through the file line by line with open("demofile.txt") as f: for x in f: print(x)

By default, read() returns the whole text. You can specify how many characters to return (e.g., read(5)).

You can return one line using readline(). To read the whole file efficiently without loading it all into memory at once, loop through the file object line by line.

!
Pro Tip: Never use read() on a massive file (like 10GB) as it will crash your memory. Always loop line-by-line for line in f: for large files.

Writing to Existing Files

To write to an existing file, you must add a parameter to the open() function:

  • "a" - Append: Will append to the end of the file.
  • "w" - Write: Will overwrite any existing content.
Real-World Scenario
An "audit.log" uses "a" to keep a history of all events. A "config.json" uses "w" to save the current state, replacing the old state entirely.
writing.py
# Append to the end of the file with open("demofile.txt", "a") as f: f.write("Now the file has more content!") # Overwrite the entire file with open("demofile.txt", "w") as f: f.write("Woops! I have deleted the content!") # Open and read to verify with open("demofile.txt") as f: print(f.read())

Create & Delete Files

os_ops.py
# To create a new file, use open() with: # "x" - Create (Error if exists) # "w" or "a" - Creates if not exists f = open("myfile.txt", "x") # To delete a file, import OS module import os # Check if file exists before deleting if os.path.exists("demofile.txt"): os.remove("demofile.txt") else: print("The file does not exist") # Remove an empty folder os.rmdir("myfolder")

To create a new file in Python, use the open() method with "x", "a", or "w".

To delete a file, you must import the OS module and run its os.remove() function. Always check if the file exists to avoid errors.

To delete an entire folder, use os.rmdir(). Note that you can only remove empty folders.

!
Pro Tip: os.rmdir() fails if the folder has files inside. To delete a non-empty folder, use the shutil module: shutil.rmtree('folder_name').
